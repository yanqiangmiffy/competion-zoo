# competion_zoo
机器学习比赛的一些通用方法和模板

## 程序说明

- `baseline.py`：可以快速比较下sklearn的各种学习器在数据集上的效果

- `average_demo.py`:平均结果：`finalpred=(pred1+pred2+pred3)/3`

- `weight_demo.py`:结果加权求和：`finalpred=(pred1*0.3+pred2*0.3+pred3*0.4)`

- `stacking_demo.py`:模型集成-stacking
